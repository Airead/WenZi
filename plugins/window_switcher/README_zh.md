# Window Switcher

通过启动器在所有应用的窗口之间快速切换（前缀：`w`）。

## 功能

### 窗口列表

显示所有可见窗口的标题、应用名称和应用图标（32×32，缓存至磁盘）。

### 模糊搜索

输入关键词，按窗口标题或应用名称进行模糊匹配过滤。

### 操作

| 按键 | 操作 |
|------|------|
| `Enter` | 聚焦选中的窗口 |
| `Cmd+Enter` | 关闭选中的窗口 |

### 图标缓存

通过 AppKit 提取应用图标并转换为 PNG 格式，缓存至磁盘，同时使用内存缓存层确保后续查找即时显示。

## 使用方法

1. 打开 WenZi 启动器
2. 输入 `w` 查看所有打开的窗口
3. 继续输入以按窗口标题或应用名称过滤
4. 按 Enter 聚焦窗口，或按 Cmd+Enter 关闭窗口

## 要求

- WenZi ≥ 0.1.14
