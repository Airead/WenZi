# Menu Search

搜索并触发前台应用和 WenZi 自身的菜单项（前缀：`m`）。

## 功能

### 双菜单搜索

同时搜索两个菜单源：
- **前台应用** — 通过 macOS 辅助功能 API 扫描当前活跃应用的菜单栏
- **WenZi** — 扫描 WenZi 自身的状态栏菜单

### 模糊匹配

同时匹配菜单项标题和完整菜单路径（如 `文件 > 导出 > PDF`），结果按相关度排序。

### 丰富展示

每条结果显示：
- 菜单项标题
- 完整菜单路径和键盘快捷键（如有）
- WenZi 菜单项标注 `WenZi` 标签
- 已启用的菜单项显示勾选标记

### 操作

| 按键 | 操作 |
|------|------|
| `Enter` | 触发选中的菜单项 |

## 使用方法

1. 打开 WenZi 启动器
2. 输入 `m` 加搜索关键词（如 `m export`）
3. 浏览结果 — 应用菜单在前，WenZi 菜单在后
4. 按 Enter 触发选中项

## 要求

- WenZi ≥ 0.1.14
- 需要辅助功能权限（系统设置 → 隐私与安全性 → 辅助功能）
