# Crypto Prices

通过币安合约 API 在菜单栏实时显示加密货币价格。

## 功能

### 菜单栏显示

在 macOS 菜单栏直接显示当前 BTC 价格，每 5 秒自动更新。点击展开下拉菜单查看所有跟踪的加密货币价格。

### 跟踪币种

| 币种 | 交易对 |
|------|--------|
| BTC | BTCUSDT |
| ETH | ETHUSDT |
| BNB | BNBUSDT |
| SOL | SOLUSDT |
| LINK | LINKUSDT |

### 智能价格格式化

根据价格大小自动选择合适的精度：
- ≥ 10,000 → 无小数（如 `84521`）
- ≥ 1,000 → 1 位小数（如 `1843.2`）
- ≥ 1 → 2 位小数（如 `13.45`）
- < 1 → 4 位小数（如 `0.0042`）

### 快速交易

点击下拉菜单中的任意币种，可在默认浏览器中打开对应的币安交易页面（Pro 布局）。

## 使用方法

安装并启用后，插件会自动开始获取价格。BTC 价格显示在菜单栏，点击可查看所有跟踪币种。

如果 API 不可达，菜单栏会显示 `--`，直到下次成功获取。

## 要求

- WenZi ≥ 0.1.12
- 需要网络连接（币安合约 API）
